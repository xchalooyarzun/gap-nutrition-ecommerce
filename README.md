# 🛒 GAP Nutrition - Ecommerce Demo

Ecommerce de suplementos deportivos desarrollado con **React + TypeScript**.  
Este proyecto simula una tienda online con catálogo de productos, filtros, búsqueda y carrito persistente.

El objetivo del proyecto es demostrar habilidades de desarrollo frontend moderno y arquitectura de componentes utilizando React.

- Filtros por categoría
- Agregar productos al carrito

## 🛠 Instalación local

Clonar el repositorio:


git clone https://github.com/TU-USUARIO/gap-nutrition-ecommerce.git


Entrar al proyecto:


cd gap-nutrition-ecommerce


Instalar dependencias:


npm install


Ejecutar servidor:


npm run dev
